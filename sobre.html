Do que se trata o projeto ?

O Projeto visa a construção de um esqueleto que posteriormente receberá a implementação de um prompt em JavaScript para poder gerar algumas imagens baseadas no que o usúario solicitou, a parte do index faz uma apresentação e explica as funcionalidades do sistema, a pagina de galeria serve como um mostrador das imagens que já foram geradas.
